### EN

```
Slackline.Ova is an offline-first slackline map for slackliners. Find every public slackline in the Czech Republic and 7810 lines worldwide — anchor points, lengths, heights, descriptions, ratings, parking spots.

WHAT IT DOES

★ Offline map — all slacklines downloaded to your phone from the first launch. Works just as well in the field without signal as it does at home.

★ Two data sources — slack.cz (239 lines from the Czech community, build-time updated) + slackmap.com (7810 lines worldwide). You can filter by source.

★ Three map styles — satellite, outdoor terrain, OSM. Switch in settings.

★ Search + filters — name, region, sector, country. Cascading filter automatically offers only relevant combinations.

★ Bottom sheet UX like Google Maps — map fills the whole screen, slackline list slides up from the bottom with your thumb. Three sizes (map large / half-half / list large).

★ Table list — length, height, rating stacked, easy to compare.

★ GPS — tap the logo to center map on your location. Optional, everything else works without GPS.

★ Czech and English — switch in settings.

★ Dark mode — automatic per system preference.

DATA SOURCES

Slackline data comes from two public sources:
- slack.cz — Czech slackline community database, manually maintained
- slackmap.com — international registry (ISA — International Slackline Association)

The app only reads the data, does not contribute back. When you find a new line, submit it via slack.cz or slackmap.com.

PRIVACY

★ No accounts
★ No tracking
★ No ads
★ No analytics
★ No third-party data sharing

Everything happens on your phone. GPS location is used locally only, never transmitted anywhere. Privacy policy: https://zemanektomas.github.io/slack-ova-mobile/privacy

OPEN SOURCE

Apache 2.0 license. Code and issue tracker on GitHub:
https://github.com/zemanektomas/slack-ova-mobile

Pull requests welcome.

COMMUNITY

Slackline.Ova originates from the Ostrava slackline scene. Brand identity is Ostrava-themed, but data is global — the app is useful anywhere slackliners walk the line.
```

## Marketing screenshot checklist

Pro každou platformu potřebuješ minimálně 2, ideálně 4-8 screenshotů. Doporučené záběry:

1. **Home screen — mapa + sheet half** — ukáže primární UX
2. **Map + bottom sheet expanded** — zoom na slackliny v ČR (Brno / Praha / Ostrava region)
3. **Inline detail rozbalený** — ukáže parametry slackline
4. **Settings sheet otevřený** — ukáže nastavitelnost (jazyk, podklad mapy, zdroj)
5. **Filter sheet** — cascading státy/regiony
6. **Dark mode** — totéž v dark theme
7. **GPS aktivní** — uživatelská poloha + halo
