# Ostravak — průvodce stylem pro Slackline.Ova

Tenhle soubor je **fonický a stylistický návod** pro psaní v projektu Slackline.Ova:
- README hlavičky a vtipné poznámky
- Commit messages s lehkostí (ne všechny — viz „Co psát spisovně")
- Issue titles a popisy
- Notes pro testery, release notes

Cíl: nepsat to jako korporát, ale jako parta lidí co spolu chodili po pradelnich šňurach. Brand Slackline.Ova = ostravská komunita. Když to budu psát spisovně, je to nudný a každý hned pozná, že to dělala AI. Ostravský dialekt + lehký humor = jeden ze způsobů, jak udržet osobnost projektu.

---

## Co psát spisovně, co po ostravsku

| Typ | Jazyk |
|---|---|
| **Architektonické dokumenty** (CLAUDE.md, ADR) | spisovně |
| **API kontrakt / endpoint dokumentace** | spisovně |
| **README intro + technické sekce** | spisovně |
| **README "co umí" / "jak nainstalovat"** | po ostravsku jde, s mírou |
| **Commit messages — technické** (CI, refactor, fix) | spisovně |
| **Commit messages — feature releases** | po ostravsku **lze** v subject line, spisovně v těle |
| **Issue title** | po ostravsku jde |
| **Issue body** | spisovně (musí to číst i AI / contributor) |
| **Release notes** | mix — top-line po ostravsku, changelog spisovně |
| **WhatsApp / chat / Slack zprávy** | po ostravsku, ano |
| **Komentáře v kódu** | spisovně, žádné nářečí |

Pravidlo č. 1: **Když na tom záleží srozumitelnost (kód, kontrakty, debug), píšu spisovně.** Pravidlo č. 2: **Když to může být lehčí (README intro, release notes hlavička, story-style commit), můžu si po ostravsku oddychnout.** Pravidlo č. 3: **Nikdy nepoužívám obě varianty v jednom odstavci** — působí to schizofrenně.

## Fonetické posuny (jak slova zní ostravsky)

| Spisovně | Po ostravsku | Pozn. |
|---|---|---|
| dlouhé á, í, ú, ý | krátké a, i, u, y | „**ma**lo víc, **ka**žda lajna" (ne „má, každá") |
| sloveso **být**, 1. os. plurálu | **sem / sme** | „já **sem** lez", „**sme** v lese" |
| **ě**, **j** v koncovkách | někdy mizí | „přebal**ova**ni" (ne „přebal**ová**ní"), „čty**ry**", „dru**hy**" |
| **-l** v minulém čase | krácení o -l | „udě**l**al" → „udě**l**", „byl" → „**byl**" (zachováno), „**čet**" (četl) |
| zájmena **on, ona** | **vůn, vůna** (vzácně) / **on, ona** | běžně se nemění, ale **mne** místo **mě** dost |
| **ž** v koncovce | **š** někdy | „kaž**ž**dy" → „kaž**de**ho"... drží se |
| **r** + **ž** | **ř** krátké | normálně, není ostravský specifik |

**Místo „já" / „on":** často „**ja**" (krátké), „**vůn**" / „**vůna**" (vzácně, spíš lašsky).

**Slovesa v minulém čase, 1. os. mužský rod:**
- „udě**lal sem**" / „**ude**lal sem"
- „**na**lez sem" — krátké
- „**byl** sem", „**chod**il sem"

**Slovesa 3. os. plurálu — koncovka -aju, -uju:**
- „lidi cho**daju**" (chodí)
- „kluci táh**aju**" (tahají)
- „**dělaju**" (dělají)
- „lepi**ču** mez**i**tim sle**piju**" / „lepi**ly**" → „**slepi**ly"

## Vzorové věty z předlohy (denik.ostravaka.cz)

Z textu na denik.ostravaka.cz, autentický **silně ostravský** styl. Použité jako referenční korpus pro „jak to autenticky zní". Pro náš projekt to budem mírnit — silnější fonetika („něvim → nevim", „enem → jen") je OK občas, ne pořád.

```
Včera sem byl nakupiť v tym novym Kaufu, bo v ledničce už byla vidět
delši dobu zadni stěna a mama k temu tež přyhodila menši seznam.
```
→ Vzor: **otevírací věta s konkrétním detailem** + ironie nad nedostatkem v ledničce + zapojení mámy jako další viník.

```
V kecafonu pry, kupiš mi enem paru věcy, ja teho moc něpotřebuju.
Jak sem to slyšel, rači sem sy vzal aštverku.
```
→ Vzor: **mama říká „jen pár věcí" a syn očekává nákladní vůz**. Sebekriticky.

```
Drapnul sem vuzek a šel bludiť mezy regalami, kaj eště nědavno
jezdily autobusy.
```
→ Vzor: **konkrétní akce („drapnul sem vuzek")** + ironický geografický popis (hypermarket = "kaj jezdily autobusy" = nesmyslně velký).

```
Tuž do štajgrove řytě, to ma byť nahrada za mě? To musela dluho
hledať, enem co je pravda.
```
→ Vzor: **silná zaklená reakce + ironický kompliment soupeři**. Tohle je extrém, použít jen v silně neformálním kontextu.

```
Naraz spoza regalu vylez taki ulizany černovlasy pajtaš s culikem
a hrnul se k ni.
```
→ Vzor: **vícenásobný adjektivum** („taki ulizany černovlasy") + nedostatek lichotek jako charakterizace.

```
Lepil se na ňu jak snopel na nehet.
```
→ **Bonus**: lidové přirovnání. Tyhle perly použijeme spíš výjimečně.

```
Tuž zpatki. Ten tyden sem sy fajně vypraznil palu a ta chata tež
něvyhořela, tuž je všecko v rychtyku.
```
→ Vzor: **návrat z dovolené** = „vypraznil palu" (vyprázdnil hlavu) + suchý fakt že „chata něvyhořela".

```
Karel chvilu chodil sem a tam a cosyk mumlal, ale nakoněc v nim
zvitězyl ten lepši Karel.
```
→ Vzor: **„ten lepší X v něm zvítězil"** — typický idiom pro rozhodnutí proti instinktu. Hodí se pro release notes: „V CI pipeline nakoněc zvitězyl ten lepši regex" / „V architekture zvitězyl ten lepši Karel — read-only mobil."

```
Bechnul sebu do lužka a pry, že zytra valime, bo kola su jak nove.
```
→ Vzor: **únava + závěrečný plán na zítra**. Použitelné pro release notes uzavření: „Bechnul sem release ven a zytra valime na crossings + login."

```
Smiřyl sem se s tym, že dalši vylety stravim ve svojich jetych
tenyskach, syce tež od ťamanu, ale mam už ich třeti rok a furt
služa dobře.
```
→ Vzor: **smířenost s nedokonalostí** + obrana („služa dobře furt"). Použitelné pro dokumentaci known issues: „Mapy.cz tiles vyžaduji signal — syce furt služa dobře, ale offline MBTiles je v F4."

## Doplnění k fonetice (z denik.ostravaka korpusu)

| Spisovně | Po ostravsku | Pravidlo |
|---|---|---|
| -**ý** koncovka | -**y** | „nov**ý** Kauf" → „nov**y** Kauf"; „chudobn**ý**" → „chudobn**y**" |
| -**í** koncovka | -**y** nebo -**i** | „mineralek a trvanlivych mlik" („trvanlivých" → „trvanlivych") |
| **ji**- na začátku | **i**- | „**ji**ž" → „**iž**" / nemizí |
| -t koncovka infinitivu | -**ť** | „mluvi**t**" → „mluvi**ť**", „nakupi**t**" → „nakupi**ť**" |
| **ne**- prefix záporu | **ně**- | „**ne**vim" → „**ně**vim", „**ne**chce" → „**ně**chce" |
| **vy**- prefix | **vy**- (zůstává) | „**vy**rážim", „**vy**pravil" |
| **vz**- prefix | **z**- (často) | „vzpomínám" → „spomínám / spoměl" |
| -**l** v min. čase mužsky | -**l** ale palatalizace | „pomohl" → „pomoh"; „padl" → „pad" |
| -**ž** koncovka | -**š** | „mu**ž**" → většinou „mu**ž**", ale: „už**ž**" → „už", "tež" místo „též" |
| **mě**, **tě** | **mě**, **tě** + někdy **mě**, **tě**, **něm**, **němu** | drží se, ale dlouhé „mně" se zkracuje |

## Šablony pro každodenní použití

### Commit message subject line (Conventional Commits + ostravsky)

**Spisovně** (default pro většinu commits):
```
feat(filter): search + cascading state/region/sector filters
fix(ci): anchor signing config regex to release block only
docs(readme): add download section
```

**Po ostravsku** (jen občas, ne kažou věc):
```
feat(filter): hledame v 8000 lajnach a nezasekneme se ani na CZ
feat(brand): logo se prevleklo, už nejsme Slackline CZ, sme Slackline.Ova
chore: lepime CI nazpatek, ten regex byl po slivovici
```

**Pravidlo**: subject line po ostravsku, **body vždy spisovně** (musí to dát smysl AI revieweru, contributorovi, budoucímu já). Vtip patří do názvu, fakta do popisu.

### Issue titles

**Spisovně:**
```
#21 Google OAuth flow (expo-auth-session)
#20 Offline MBTiles for ČR (zoom 6-14)
```

**Po ostravsku:**
```
#20 Stahnout si mapu do telefonu — ať to chodi i tam kde neni signal
#21 Loginovat se přes Google — ať vime, kdo nam tam chodi
```

V issue body se zase vrátit ke spisovnému jazyku.

### Release notes hlavička

**Spisovně:**
```
## v0.3.0 — Slackline.Ova rebrand + search/filter + GPS auto-detect
```

**Po ostravsku (volitelná verze):**
```
## v0.3.0 — Sme Slackline.Ova a hledame, co kde visi
```

### README intro paragraf

**Současný (spisovně):**
> Slackline.Ova — offline-first mobilní aplikace pro slackliners. Ukazuje mapu se slacklines, seznam viditelných linií filtrovaný podle výřezu mapy.

**Alternativa po ostravsku (volitelné, jako "lehčí úvod"):**
> Slackline.Ova ti ukaže kdy a kde visi nejaka lajna v okoli a jak je dluha. Stači mit telefon a chodit. Internet nepotřebuje, mapa i lajny mas v batohu i v lese kde neni signal.

### Funkční popis (například „Stažení")

**Spisovně:**
> 1. Otevři odkaz výše v Chrome na Androidu
> 2. Klikni na `slackline-ova-<verze>.apk` v sekci Assets — stáhne se ~130 MB

**Alternativa po ostravsku:**
> 1. Otevři ten link v Chrome (Android telefon)
> 2. Klikne se ti na `slackline-ova-<verze>.apk` v Assets — stahne se ti taka kupa bitu, 130 MB

Vzor: **otevírací věta = překvapení + ironie nad sebou samým**. Funguje pro release notes, blog post, story bočného sloupce.

```
Paru roku sem chodil kolem takych divnych individui v komeňaku a pozoroval co to robi
za hitparady na pradelnich šňurach.
```

Vzor: **konkrétní detail + ironický pohled outsidera**. „divnych individui" je sebezačerňující, „hitparady" je hyperbola.

```
S udernu směnu co sme o sebe zakopli na Čupku chystame 100m špagatu.
```

Vzor: **kolektivní podmět („sme") + obyčejné setkání („zakopli o sebe") + technický cíl („100m špagatu")**. Funguje pro changelog: „**S Petrom sme se domluvili na /mapa/data — bude to autoritativni zdroj.**"

```
Tož kolik sily zbyde na špagat, sem zvědavy.
```

Vzor: **suchý vtip nad vlastní limitací** („Tož kolik baterky zbyde na slackmap, sem zvědavy.")

```
Jo a ve středu valime tahat Satiny, tak zduř!
```

Vzor: **závěrečný plug pro budoucí akci** + zvací výkřik. Použitelné pro release notes konec: „Jo a v0.4.0 bude crossings + login, tak držte palce a hlavně testujte!"

## Co NE-používat

- **Náhodné anglikalismy v ostravském textu** — „awesome app" se nezní dobře. Buď celé česky, nebo celé anglicky.
- **Emoji v textu**, pokud TZ explicitně nepožádá. Ostravak je suchý humor, ne instagrammy ferment.
- **Mix češtinu+ostravak v jedné větě** — „udělal jsem feature, kde špagat **šněruje**" je trapné. Buď „udělal jsem search funkcionalitu" nebo „splašil sem search, hleda v 8000 lajnach".
- **Pseudo-archaizmy** typu „kéž by to fungovalo, milý vandráku". Není to historický text, jen folklorně-mluvený.
- **Ostravský jazyk pro chyby/erroru** — uživatel co dostane crash hlášku „Něco sa zaseklo, mrkni do logu" je naštvaný 2× tolik. Erroru spisovně, neutrálně.

## Test: pozná to AI / outsider?

Když text projde tímhle filtrem:
1. **Čte se to lehce nahlas?** (Ostravak je mluvený, ne psaný — testuj ústně.)
2. **Ironie / podceňování sebe** je tam, ale ne agresivní vůči jinému?
3. **Konkrétní detail** (místo, čas, předmět) zakotvuje text v realitě?
4. **Subjekt „sme / sem"** je tam? (Bez něj to není ostravak.)
5. **Žádný emoji, žádný corporate filler** („we are excited to announce")?

Když všech 5 splňuje, jsi v rytmu. Když ne, vrať se ke spisovné.

## Použití tohoto souboru

- Když chystáš PR / commit / release notes a chceš to napsat lehce, mrkni sem na slovník a vzory.
- Tenhle soubor je **living document** — když objevíš nový obrat, který se hodí, dopiš ho do slovníku.
- Když ti to nepřijde přirozené (například jsi z Prahy a píšeš to "uměle"), **zůstaň spisovně**. Falešný ostravak je horší než žádný.
